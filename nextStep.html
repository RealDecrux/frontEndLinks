Geral

Quando você clica duas vezes no ícone WorldWideWeb, o aplicativo por padrão exibe sua própria página inicial, ~/WWW/default.html ou falhando que a página inicial do sistema usr/local/lib/WWW/default.html.

A partir daí, se você quiser selecionar um documento referido por uma página que está sendo exibida, basta clicar duas vezes em uma área de texto cinza. (Gray foi escolhido para não usar outras técnicas de destaque, como negrito ou itálico, que os autores podem querer usar para outras coisas. Quando a cor aparece, podemos usar a cor...)

Se você clicar duas vezes em qualquer outro lugar do documento, você selecionará uma palavra, como é de costume no NeXT.

Algumas áreas cinzentas levam a documentos inteiros, enquanto algumas trazem um documento com uma determinada área alvo destacada. Isso é chamado de âncora de destino. Ele não é cinza a menos que, por sua vez, leve a outro lugar, mas clicar duas vezes em uma âncora de destino seleciona toda a âncora.

Índices

Certos documentos são índices. Eles normalmente têm apenas uma pequena quantidade de texto. Você não pode ler o índice sozinho (pode ser muito grande). Para acessar algo através do índice, use a opção "Pesquisa de palavras-chave". A janela com o índice deve ser a janela principal (barra de título escuro) quando você usa o painel de pesquisa de palavras-chave. As regras utilizadas pelos índices para procurar correspondências variam. Diferentes índices usam técnicas diferentes e cobrem diferentes faixas de informações.

Quando uma pesquisa é bem sucedida, o resultado é uma página de referências a documentos. Você seleciona documentos dele clicando duas vezes da maneira usual.
